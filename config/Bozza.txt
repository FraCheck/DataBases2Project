Website:
	-> RegistrationPage
		Content:
			-> 3 Textboxes (user,pass,email)
			-> 1 Button to send the registration
		Functionality:
			-> Redirect to Homepage if registration succeed
			
	-> LoginForm
		Content:
			-> 2 Textboxes (user/email,pass)
			-> 1 Button to send the login
		Functionality:
			-> Redirect to Homepage if login succeed
			
	-> HomePage
		Content:
			-> Name and image of the "product of the day"
			-> Product reviews by other users
		Links:
			-> "Questionnaire of the day" --> QuestionairePage
			
	-> QuestionnairePage
		Content:
			- Section1:
				Content:
					-> Variable numbers of question labels + textboxes answers
					-> 1 Button "Next" to access Section2
				Constraints : All marketing inputs are mandatory in order to submit.
			- Section2:
				Content:
					-> Fixed numbers of inputs elements for age,sex,expertise level
					-> 2 Button "Submit", "Cancel", "Previous"
				Constraints: All statistical inputs are optional
			Functionality:
				-> Redirec to Thanks&GreetingsPage on "Submit"
	-Thanks&GreetingsPage

	PRODUCT:
		-> int id PRIMARY KEY

	REVIEWS:
		-> int id PRIMARY KEY

	QUESTIONNAIRE_USER_ANSWERS

	MARKETING_ANSWERS:
		-> int id PRIMARY KEY

	BAD_WORDS:
